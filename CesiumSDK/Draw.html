加载 3D 管道 loadPipeline ​

  • @param
参数名type描述默认值
positionArray矩形数据 ,世界坐标数组-
widthNumber管道宽度-
myDataObject自定义的数据，在监听点击事件时根据此属性获取-
PolylineVolumeStyleObject其它一些关于 PolylineVolume 的配置，参考 cesium 官方文档-
  • @returns
返回值type描述
pipelineObject返回矩形的 entity 对象。

注意

  1. 如果使用自定义图片着色器材质，需要对坐标进行插值处理，否则在一段管道中如果每一段的长度不一样，则管道的材质将无法正常显示！
  2. 注意插值精度越小性能消耗越大；具体参考 CM.BaseFn.coordinateInterpolation 方法！
js
let wgs_84Arr = [
  {
    lng: 111.17831238175873,
    lat: 24.495787476202803,
    height: 0,
  },
  {
    lng: 111.1784,
    lat: 24.4968,
    height: 0,
  },
  {
    lng: 111.2018438418352,
    lat: 24.501776466940694,
    height: 0,
  },
  {
    lng: 111.20228120412749,
    lat: 24.5028655488217,
    height: 0,
  },
];

// let cartesian = CM.Position.transformWGS84ArrayToCartesianArray(wgs_84Arr);
// //如果不使用自定义图片着色器材质只需要传入三维坐标即可，性能更优！
// const pipeline = CM.Draw.loadPipeline(cartesian, 10, {});

let cartesian = await CM.BaseFn.coordinateInterpolation(wgs_84Arr, 200);
const pipeline = CM.Draw.loadPipeline(
  cartesian,
  15,
  {},
  {
    material: new CM.Material.PipelineflowMaterialProperty({
      color: Cesium.Color.fromCssColorString("#6AE7FF"),
      speed: 1, // 速度
      image: img,
      repeat: { x: 10, y: 2 },
      direction: -1,
    }),
  }
);
