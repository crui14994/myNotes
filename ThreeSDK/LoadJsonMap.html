createMap 根据 jsonUrl 加载地图数据并解析为 3D 模型 ​

  • @params
返回值type描述
urlString地图 GeoJson 数据的 URL
mapConfigObject地图的配置项，包括深度、线条材质、多边形材质等
mapConfig-配置项 ​
返回值type描述
depthNumber地图拉伸高度
extrudeGeometryConfigObject地图的拉伸配置项，参考 ExtrudeGeometry 配置
lineMaterialObjectthree 创建的线条材质对象 ，参考 LineBasicMaterial 配置
sideMaterialObjectthree 创建的侧边拉伸材质对象，参考 Material 配置
polygonMaterialObjectthree 创建的区快材质对象，参考 Material 配置
  • @returns
返回值type描述
mapObjectthree 创建的 Mesh 对象
js
const lineMaterial = new THREE.LineBasicMaterial({ color: new THREE.Color("#FFFFFF") });
const sideMaterial = new THREE.MeshBasicMaterial({
  color: new THREE.Color("#2E2E2E"),
});
const texture = new THREE.TextureLoader().load("Three/images/zitong.png");
texture.wrapS = THREE.RepeatWrapping;
texture.wrapT = THREE.RepeatWrapping;
texture.repeat.set(1, 1);

const polygonMaterial = new THREE.MeshBasicMaterial({
  map: texture,
});
let map = await MT.LoadMap.createMap("Three/data/zitong.json", {
  depth: 0.2,
  lineMaterial: lineMaterial,
  sideMaterial: sideMaterial,
  polygonMaterial: polygonMaterial,
});
console.log(map);
scene.add(map);

loadPolygon 根据坐标数组解析为 3D 模型 ​

  • @params
返回值type描述
coordinatesArray二维坐标数组
mapConfigObject地图的配置项，包括深度、线条材质、多边形材质等
mapConfig-配置项 ​
返回值type描述
depthNumber拉伸高度
extrudeGeometryConfigObject拉伸配置项，参考 ExtrudeGeometry 配置
lineMaterialObjectthree 创建的线条材质对象 ，参考 LineBasicMaterial 配置
sideMaterialObjectthree 创建的侧边拉伸材质对象，参考 Material 配置
polygonMaterialObjectthree 创建的区快材质对象，参考 Material 配置
  • @returns
返回值type描述
polygonObjectthree 创建的 Mesh 对象
js
let coordinates = [
  [
    [105.459198, 31.526955],
    [105.454726, 31.528591],
    [105.453664, 31.529683],
    [105.452829, 31.531968],
    // ...
    [105.456284, 31.508389],
    [105.459648, 31.509414],
    [105.461505, 31.510743],
    [105.459198, 31.526955],
  ],
];

const texture = new THREE.TextureLoader().load("Three/data/test.jpg");
texture.wrapS = THREE.RepeatWrapping;
texture.wrapT = THREE.RepeatWrapping;
texture.repeat.set(1, 1);

const polygonMaterial = new THREE.MeshBasicMaterial({
  map: texture,
});
let polygon = MT.LoadMap.loadPolygon(coordinates, {
  depth: 0.1,
  polygonMaterial: polygonMaterial,
});

polygon.position.y = -0.05;
scene.add(polygon);
