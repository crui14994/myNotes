rotateCircle 创建并配置一个包含正反向旋转圆盘底座 ​

  • @param
参数名type描述默认值
positionArray圆盘组的三维坐标[0,0,0]
scaleArray圆盘缩放比例[1,1]
rotationSpeedNumber云的初始速度0.05
heightNumber圆盘组基准高度0
colorSting圆盘材质颜色#FFFFFF
opacityNumber圆盘材质透明度0.2
js
let rotateCircle = MT.LoadMesh.rotateCircle({
  scale: [2.5, 2.5], // 大小圆盘缩放比例
  // color: new THREE.Color('rgb(255, 0, 0)')
});
scene.add(rotateCircle);

flowingLight 创建并返回一个沿自定义路径流动的光带效果的三维网格对象 ​

  • @param
参数名type描述默认值
pointsArray三维点坐标数组[]
proportionNumber光带长度占路径的比例 (0~1)0.07
sizeNumber光带尺寸0.003
speedNumber光带流动速度0.002
heightNumber光带在 Z 轴方向的高度偏移0.09
colorSting线条颜色#FFFFFF
activeColorSting流光颜色yellow
js
let coordinates = [
  [105.459198, 31.526955],
  [105.454726, 31.528591],
  [105.453664, 31.529683],
  // .............
  [105.452829, 31.531968],
  [105.448316, 31.533012],
  [105.446875, 31.534004],
  [105.445453, 31.536351],
];
//需要将经纬度转为三维坐标
let points = coordinates.map((item) => {
  item = MT.Position.coordinateTransformation(item);
  return new THREE.Vector3(item[0], item[1], 0);
});

// 创建一个沿自定义路径流动的光带效果
let curveObject = MT.LoadMesh.flowingLight(points, {
  height: 0.1,
  size: 0.004,
});
/**添加到场景或其它对象中 */
scene.add(curveObject);

createLabel 创建并配置一个 CSS2D 标签对象 ​

  • @param
参数名type描述默认值
idString静态标签元素的 DOM ID（isCreateElement 为 false 时生效）''
positionArray标签在 3D 空间中的坐标[x, y, z][0, 0, 0]
isCreateElementBoolean是否动态创建 DOM 元素（多标签建议开启，单标签建议关闭）false
elementHTMLElement动态创建模式时传入的 DOM 元素引用-

有两种方式创建标签对象(注意 css 样式对标签显示的影响)：

  1. 通过 DOM ID 创建标签对象,适合页面只显示一个标签的情况,并且此 DOM 对象是在 html 中定义好的!
js
let label = MT.LoadMesh.createLabel({
  id: "labelId",
  position: [0, 1, 0],
});
scene.add(label);

  1. 通过 isCreateElement 为 true 开启动态使用 js 创建标签对象,适合页面显示多个标签的情况;开启后通过 element 传入!
js
let element = document.createElement("div");
element.textContent = "测试标签2";
element.className = "label-test";

let label = MT.LoadMesh.createLabel({
  isCreateElement: true,
  element: element,
  position: [0, 1, 0],
});
scene.add(label);

createPoint3D 创建带有标签和动画的 3D 点对象 ​

需要特别注意光照对 3D 点位的影响,没有光照可能会造成颜色暗淡的情况

  • @param
参数名type描述默认值
sizeNumber点的大小0.1
colorString点的颜色值#C59115
floatingRangeNumber浮动动画的垂直幅度0.2
positionArray点物体的三维坐标[0,0,0]
showLabelBoolean是否显示标签true
labelAnimationBoolean标签是否跟随浮动true
labelIdString静态标签元素的 DOM labelElement 为 null 时生效）''
labelPositionArray标签相对于父对象的三维位置[0,0,0]
labelElementHTMLElement动态创建模式时传入的 DOM 元素引用null

有两种方式创建 3D 点位(注意 css 样式对标签显示的影响)：

  1. 通过 DOM ID 创建,适合页面只显示一个标签的情况,并且此 DOM 对象是在 html 中定义好的!
js
// 添加方向光创造阴影
const directionalLight = new THREE.DirectionalLight(0xffffff, 1);
directionalLight.position.set(1, 2, 3);
scene.add(directionalLight);
//设置光照
const light = new THREE.AmbientLight(0xffffff, 0.8); // soft white light
scene.add(light);

let point3D = MT.LoadMesh.createPoint3D({
  labelId: "ThreeLabel",
  labelPosition: [0, -0.4, 0],
  labelAnimation: true,
  position: [-0.3, 0.1, 0.4],
  color: "#FD6AAF",
});
scene.add(point3D);
