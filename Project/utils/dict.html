概述 ​

该文件提供了字典相关的常用方法，主要用于获取字典项及其标签。

这些方法在项目中被广泛使用，特别是 getDictArr 和 getDictLabel 需要全局挂载以便在组件中方便调用。

文件来源：/src/utils/dict.js

全局挂载 ​

为了在所有组件中直接使用 getDictArr 和 getDictLabel，需要在 main.js 中进行全局挂载：

js
import { getDictArr, getDictLabel } from "@utils/dict.js";
app.config.globalProperties.getDictArr = getDictArr;
app.config.globalProperties.getDictLabel = getDictLabel;

getDictArr 根据字典类型获取字典项（可传入多个） ​

功能描述

  • 动态获取字典项，优先从 store 中查找，若未找到则通过接口获取。

  • 将字典项以字典类型为键存储在响应式对象中，返回该响应式对象的引用，以便在其他地方响应式地访问更新后的字典项。

使用示例

点我查看代码
html
<template>
  ...
  <!-- sys_job_status 已经是响应式对象 -->
  <div v-for="item in sys_job_status" :key="item.value">{{ item.name }}</div>
  ....
</template>

<script setup>
  import { getCurrentInstance } from "vue";
  // 需要在main.js中全局挂载getDictArr方法
  const { proxy } = getCurrentInstance();
  const { sys_job_group, sys_job_status } = proxy.getDictArr("sys_job_group", "sys_job_status");


  //特别注意：
  // 在js中sys_job_group和sys_job_status都是响应式对象，所以需要使用.value获取值
  console.log(sys_job_group.value, sys_job_status.value);

  // 在html中或者是将字典项传入组件中以html的形式渲染则不能加.value，否则会失去响应式；如flexForm组件下拉选项配置的options就不能加.value！
  const itemArr = ref([
    ...
    {
        type: 'select',
        label: '任务分组：',
        prop: 'jobGroup',
        disabled: false,
        placeholder: '请选择任务分组',
        rules: ['required'], // 验证规则
        flexWidth: '50%',
        multiple: false, //是否多选
        options: sys_job_group,
    },
    ...
  ])

  //如果是在计算属性中，因为本身计算属性就可以看作是响应式的，会动态计算，所以在计算属性中需要加.value
  const itemArr = computed(() => {
    return [
      ...
    {
        type: 'select',
        label: '任务分组：',
        prop: 'jobGroup',
        disabled: false,
        placeholder: '请选择任务分组',
        rules: ['required'], // 验证规则
        flexWidth: '50%',
        multiple: false, //是否多选
        options: sys_job_group.value,
    },
    ...
    ]
  })
</script>

getDictLabel 根据字典类型和值获取字典项的标签(使用此方法要先确保已经获取过字典) ​

功能描述

  • 根据给定的字典类型和值，返回对应的字典项标签

使用示例

1. 在 template 中使用 ​
html
<template>
  ...
  <!--  在html中使用此方法不必在js中通过proxy 引入；可直接调用 -->
  <div>字典：{{getDictLabel("sys_job_status", 1)}}</div>
  ....
</template>

2. 在 script 中使用 ​
js
import { getCurrentInstance } from "vue";
// 需要在main.js中全局挂载getDictLabel方法
const { proxy } = getCurrentInstance();

let label = ref("");

function getDictLabel() {
  label.value = proxy.getDictLabel("sys_job_status", 1);
}
