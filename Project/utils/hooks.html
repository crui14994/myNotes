formatImgUrl 图片地址处理 ​

从后端获取图片地址不能直接显示在图片组件中，需要使用 formatImgUrl 函数处理图片地址拼接上前缀才能正常显示

js
<el-image :src="formatImgUrl(nowClickData.imgUrl)" fit="cover"></el-image>

const { formatImgUrl } = useImgUrl();

useECharts Echarts 封装统一处理初始化和监听窗口变化 ​

点我查看
vue
<template>
  <div ref="chartRef" style="width: 100%; height: 400px"></div>
</template>

<script setup>
import { ref ,watch} from "vue";
import { useECharts } from "@/hooks/useECharts";

const props = defineProps({
  echartsData: {
    type: Object,
    required: true,
    default: () => {},
  },
});

const chartRef = ref(null);

// 图表配置
const chartOptions = {
  xAxis: {
    type: 'category',
    data: ['A', 'B', 'C']
  },
  yAxis: {
    type: 'value'
  },
  series: [{
    name: '示例',
    type: 'bar',
    data: [10, 20, 30]
  }]
};



// 调用 Hook 初始化图表
const { chart, updateOptions } = useECharts(chartRef, chartOptions);


/**处理数据 */
const handleEchartsData = () => {
  chartOptions = .... //根据props.echartsData处理数据

   updateOptions(chartOptions);
}

watch(
  () => props.echartsData,
  async (newValue) => {
    if (newValue && updateOptions) {
      handleEchartsData();
    }
  }
);

</script>
