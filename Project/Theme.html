主题颜色或其它变量 ​

所有颜色或其它变量均定义在这个/src/styles/variables.scss文件中！

注意：编写页面和封装组件的时候必须从此取变量，达到统一规范化！如：内外边距，颜色等！

js
//通用默认内边距
$default-padding: 20px;

//通用默认外边距
$default-margin: 16px;

//侧边导航背景色
$silder-bar-bg-color: #1b2322;

//内容页最外层背景色
$default-main-bg-color: #f6f7f8;

//GIS地图的主色调
$default-gis-color: #105E4A;

自定义ElemrntUiPlus主题颜色 ​

重写主题颜色

通过 @forward 指令，重写了 Element Plus 的主题颜色配置

  • white:

    #ffffff

  • black:

    #000000

  • primary:

    base: #08A365 主色，通常用于主要按钮、链接等元素。

  • success:

    base: #1dc779 成功色，通常用于成功状态提示。

  • warning:

    base: #ffb302 警告色，通常用于警告状态提示。

  • danger:

    base: #e26237 危险色，通常用于危险状态提示。

  • info:

    base: #cf4444 错误色，通常用于错误状态提示。

    • error:

    base: #909399 中性色。

在组件中如何使用主题色 ​

主题颜色修改

下方代码中primary字段可根据上面自定义的主题颜色（primary，success，warning，danger，error）进行修改；

js
/* 基础颜色white */
background: var(--el-color-white);

/* 基础颜色（不透明） */
background: var(--el-color-primary);

关于自定义颜色透明度 ​

  1. 自己定义的颜色变量
js
//如果需要使用透明度，则需要使用rgba()函数，如：
background-color: rgba($default-gis-color, 0.5);

//注意：$default-gis-color必须是十六进制颜色值；elementuiPlus的颜色变量不能使用此方式

  1. elementuiPlus的颜色变量
js
//如果要使用主色的透明度，则使用color-mix()函数，如：
background-color:color-mix(in srgb, var(--el-color-primary), transparent 70%);

//注意：后面的百分比与实际透明度是相反的，如70%，则实际透明度为0.3
