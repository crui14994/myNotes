分页器基本使用 ​

无需引入 直接使用

  1. 传入pageNum ,pageSize, 和total 接收一个分页改变事件 paginationChangeFn=”paginationChangeFn“

    javascript
        <Pagination
      v-model:current-page="pageInfor.pageNum"
      v-model:page-size="pageInfor.pageSize"
      :total="pageInfor.totalElements"
      @paginationChangeFn="paginationChangeFn"
    ></Pagination>
	//可以直接 @paginationChangeFn="getList"

    const pageInfor = ref({
        pageNum: 1,
        pageSize: 10,
        totalElements: 40,
    });

    // 分页改变
    const paginationChangeFn = (num, size) => {
      console.log(num);
      console.log(size);
      // getList()
    };

    1. 自定义内容插槽使用

      假如想自定义total相关的内容 在layout中用slot替换total即可

      javascript
          <el-pagination
      v-model:current-page="currentPage"
      v-model:page-size="pageSize"
      :page-sizes="pageSizes"
      :size="size"
      :disabled="disabled"
      :background="background"
      layout="slot,sizes, prev, pager, next"
      :total="total"
      :teleported="teleported"
      :pager-count="pageCount"
      @size-change="handleSizeChange"
      @current-change="handleCurrentChange"
      :popper-class="paginationPopperClass"
      :hide-on-single-page="hideOnSinglePage"
      @change="paginationChangeFn"
    >
    <div key="1"> 这是第一个slot-total</div>
    </el-pagination>
