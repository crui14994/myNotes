关于 Dialog 组件的说明 ​

注意事项

将不再单独封装 Dialog 组件，而是直接使用 ElementUI Plus 的 Dialog 组件！

组件公共样式修改 ​

  1. 首先在 el-dialog 标签中定义一个 class 类名base-dialog;
html
<el-dialog class="base-dialog" v-model="visible" title="Tips" width="80%" :before-close="handleClose">
  <div class="dialog-box">....</div>
  <template #footer>
    <div class="dialog-footer">
      <el-button @click="visible = false">Cancel</el-button>
      <el-button type="primary" @click="visible = false"> Confirm </el-button>
    </div>
  </template>
</el-dialog>

  1. 在全局样式中定义样式，修改 Dialog 组件的样式，如下：

文件位置：src/styles/updateStyle.scss

css
/* // Dialog组件公共样式修改 */
.base-dialog {
  overflow: hidden;
  padding: 0 !important;
  .el-dialog__header {
    padding: 0 $default-margin;
    line-height: 42px;
    height: 42px;
    background-color: var(--el-color-primary);
    .el-dialog__title {
      color: var(--el-fill-color-blank);
    }
    .el-dialog__headerbtn {
      .el-dialog__close {
        color: var(--el-fill-color-blank);
      }
    }
  }
  .el-dialog__body {
    padding: $default-margin;
  }
  .el-dialog__footer {
    padding-bottom: $default-padding;
    text-align: center;
  }
}
