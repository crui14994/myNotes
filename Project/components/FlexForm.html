FlexForm 组件 ​

注意事项

  • 组件位于项目/src/BaseComponents/FlexForm 文件目录下。
  • formData 和 itemArr 是必传参数，必须提供有效的表单数据和表单项配置。
  • 表单项的 rules 字段用于定义验证规则，规则名称应与 @/utils/RegExp 中定义的正则表达式名称一致。
使用 ​
js
// 在mian.js中全局挂载表单组件
import FlexForm from "@/BaseComponents/FlexForm/index.vue";
app.component("FlexForm", FlexForm);

Props 属性说明 ​

参数名type描述默认值必填
marginHorizontalString表单的水平方向间距'15px'否
labelPositionString表单域标签的位置 (left, right, top)'right'否
labelWidthString自定义表单域标签的宽度'auto'否
marginVerticalString表单的竖向间距'15px'否
isDisabledBoolean是否禁用全部表单，以文本方式展示false否
formDataObject表单数据对象-是
itemArrArray表单项配置数组 (具体配置参照下方表单类型及配置示例)-是
showFooterBtnBoolean是否显示下方按钮true否
btnTxtArray按钮文字["保存", "取消"]否

方法 ​

名称参数描述
submitFormformEl(表单元素引用)提交表单，验证表单数据。
resetFormformEl(表单元素引用)重置表单，清空所有字段。

在父组件中通过 ref 调用方法

js
  ...
  <FlexForm ref="formRef" :formData="formData" @saveData="saveData" :itemArr="itemArr"></FlexForm>

  ...
  <el-button type="primary" @click="handleClick">提交</el-button>
  ....

  const formRef = ref();

  const handleClick = () => {
    formRef.value.submitForm();
  };

  //建议在saveData回调函数中去处理表单验证成功后的操作
  const saveData = () => {
    console.log(formData);
  }
