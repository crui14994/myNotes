3：使用拖拽排序 ​

使用此功能必须在组件上指定sortTableId 多个表格使用排序sortTableId不能重复 保持唯一 是否开启排序由disabledDragSort决定 默认true禁用排序 设置maxHeight就可以实现拖拽滚动到底部

html
    <Table
      sortTableId="sortId"
      :tableData="tableData"
      :tableHeader="tableHeader"
      :tableMaxHeight="'500px'"
      @tableOperationFn="tableOperationFn"
      @dragSortEndFn="dragSortEndFn"
      :disabledDragSort="disabledDragSort"
    >
      </template>
    </Table>

结束拖拽的方法是 dragSortEndFn 在此方法中更新排序

javascript
/* 
    拖拽排序
    @param {Number} newIndex 新的索引
    @param {Number} oldIndex 旧的索引
 */
const dragSortEndFn = (newIndex, oldIndex) => {
  let currRow = tableData.value.splice(oldIndex, 1)[0];
  tableData.value.splice(newIndex, 0, currRow);
  tableData.value.forEach((item, index) => {
    tableData.value[index].sort = index;
  });
};
