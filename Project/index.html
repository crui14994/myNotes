项目说明 ​

基于vue3+vite+pinia+vue-router+element-plus+axios+less构建。

bash
# 安装依赖
npm install

# 启动项目,会同时启动项目和文档服务
npm run dev

# 打包项目
npm run build

# 单独启动文档服务
npm run docs
